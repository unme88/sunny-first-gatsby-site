# 2020/07/31
* Mint Card API 이름이 Mint Card To User API로 변경되었습니다.  
  * 카드를 EOA를 가진 사용자(Klip 회원 또는 비회원)에게 발행합니다.  
  * 발행 시 사용자 EOA를 사용합니다.  

* Mint Card To Klip Member API가 새로 추가되었습니다.  
  * 카드를 Klip 회원에게 발행합니다.  
  * 발행 시 Klip 회원의 실명과 전화번호를 사용합니다.  

* Send Card API 이름이 Send Card To User API로 변경되었습니다.  
  * 카드를 EOA를 가진 사용자(Klip 회원 또는 비회원)에게 전송합니다.  
  * 전송 시 사용자 EOA를 사용합니다.  

* Send Card To Klip Member API가 새로 추가되었습니다.  
  * 카드를 Klip 회원에게 전송합니다.  
  * 전송 시 Klip 회원의 실명과 전화번호를 사용합니다.  

* Send Card To User, Send Card To Klip Member API의 Response에서 `failCount` 변수명이 `fail_count`로 변경되었습니다.
