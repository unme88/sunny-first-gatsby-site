Klip REST API는 Card Minting API와 App2App API로 구성됩니다.

# Card Minting API
Card Minting API는 Klip 카드(NFT)를 발행, 전송, 삭제, 관리하는 기능 구현을 위한 REST API입니다. Card Minting API는 카드 발행을 위한 Klip 계정 로그인, 계정 관리 기능부터 카드를 발행, 전송, 관리하기 위한 API를 제공합니다.

# App2App API
App2App API는 여러분의 "BApp"을 "Klip"과 연동하는 기능 구현을 위한 API입니다. App2App API는 BApp에서 Klip 사용자 계정에 접근하여 KLAY, Klip 카드, Klip 토큰을 다른 사용자에게 전송하거나 스마트 컨트랙트를 실행하는 REST API를 제공합니다. App2App API는 REST API 및 SDK(iOS, Android, JavaScript)도 제공됩니다.

|항목|설명|비고|
|---|---|---|
| [Card Minting API](/rest-api-card-minting.md) | Klip 카드를 발행, 전송, 관리하는 기능 제공  |  |
| [App2App API](/rest-api-a2a.md) | BApp과 Klip을 연동해 Klip 카드, 토큰, KLAY 전송과 컨트랙트 실행 기능 제공  |  |

이 문서 혹은 Klip에 관한 문의는 [개발자 포럼](https://klipforum.zendesk.com)을 방문해 도움을 받으십시오.
