Klip API 튜토리얼은 Card Minting REST API 튜토리얼과 App2App REST API 튜토리얼로 구성됩니다.

# Card Minting API
Card Minting API는 Klip 카드(NFT)를 발행, 전송, 삭제, 관리하는 기능 구현을 위한 REST API입니다. Card Minting API 튜토리얼은 Klip Partners 가입과 로그인, 카드 발행, 카드 정보 조회, 카드 전송, 카드와 계정 정보를 관리하는 방법을 상세하게 안내합니다.

# App2App API
App2App API는 여러분의 "BApp"을 "Klip"과 연동하는 기능 구현을 위한 API입니다. App2App API 튜토리얼은 BApp이 Klip에 접근하기 위한 환경 설정부터 BApp에서 Klip에 요청을 보냄으로써 사용자 인증(`auth`), KLAY 전송(`send_klay`), 토큰 전송(`send_token`), 카드 전송(`send_card`), 컨트랙트 실행(`execute_contract`) 작업을 수행하는 과정을 상세하게 안내합니다. App2App API 튜토리얼은 REST API를 기준으로 안내합니다.
